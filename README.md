# Webserver
A server running on Flask uses Blueprint to manage multiple sites at once.
## Technology
frontend: js. 
backend: Python, Torch, Mongo, Mysql, GPTAPI, Flask.

## Usage
Start flask
```
pip install -r requirements.txt
flask run
```
Server run at http://*:5000/
